# Qu Liker

Qu Liker — модуль для Qu.js, который добавляет универсальные голосования и рейтинги: лайки, звёзды, emoji-реакции и thumbs vote с async-отправкой и синхронизацией состояния, с возможностью расширения типов.

- Документация: [https://qujs.ru/liker/](https://qujs.ru/liker/)
- Лицензия: MIT (см. файл `LICENSE`)
